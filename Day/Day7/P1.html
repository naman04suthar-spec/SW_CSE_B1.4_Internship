Flexbox: Flexbox layout. it a 1D layout system, to arrange and allign items in a container either in a row or a column. used for creating flexible and responsive layouts.
>> An element is turned into a flex container by setting the display: flex
>> Flex container properties:
1. display
2. Flex-direction
3. justify-content
4. align-items
5. flex-wrap
6. align-content
7. gap, row-gap, column-gap
